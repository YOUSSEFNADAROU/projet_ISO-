## 🚀 Fonctionnement Intelligent de l'Agent

### Processus Automatique

```
Question utilisateur
    ↓
Analyse intelligente (intention, domaine, urgence)
    ↓
Détection de confiance
    ↓
Si haute confiance (>50%):
  → Réponse locale expert
Sinon:
  → Appel OpenAI GPT-4 (si API key)
    ou
  → Réponse générique intelligente
    ↓
Génération de suggestions de suivi
    ↓
Réponse structurée à l'utilisateur
```

### Niveaux de Confiance

- **90-100%** 🟢 Question très claire dans un domaine connu
- **70-89%** 🟡 Question claire avec contexte identifié
- **50-69%** 🟠 Question ambiguë ou multi-domaine
- **<50%** 🔴 Utiliser OpenAI si disponible
